# Selenium Java Framework with TestNG and Maven
This is a Selenium Java framework that uses TestNG for testing and Maven for dependency management. The framework is designed to be easy to use and maintain, allowing you to quickly create and run automated tests for your web applications.

## Prerequisites
- Java Development Kit (JDK) 8 or higher
- Maven 3.6 or higher
